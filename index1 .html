git学习

  • git init 创建本地创库
  • git status 检查仓库状态q
  • git status -s 检查仓库指令
  • git add 文件名 将某个文件加入暂存区同时让他给git跟踪
  • git add . 将所有新增或已更改加入暂存区同时让他被git跟踪
  • git conmmit -m"提交说明"将暂存区中的文件入库
  • git reset HEAD 文件名将暂存区中文件移除
  • git checkout -- 文件名撤销对文件的修改（慎用，撤销后无法找回）
  • git commit -a -m "提交说明" 直接将已修改文件入库（git add / gitcommit 合二为一
